# cron

Using cron to webscrape the camera files periodically.  The `jfdcron`
file is a crontab file.  For format string see `man 5 crontab`

List existing crontab files
```
$ crontab -l
```

Remove exisitng crontab files
```
$ crontab -r
```

Edit crontab files
```
$ crontab -e
```
